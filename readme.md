## 🛠️ Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/milankathiriya/cpp_rw6.git
    ```
2. Navigate to the project directory:
    ```bash
    cd cpp_rw6
    ```
3. Compile and run the desired file:
    ```bash
    g++ array.cpp -o array && ./array
    ```
